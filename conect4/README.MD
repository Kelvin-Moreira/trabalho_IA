Para iniciar o jogo basta definir no arquivo connect4.py qual será o agente que vai jogar
se o agente desse codigo for o primeiro defina "MEU_AGENTE = AGENTE_1"
caso o agente desse codigo seja o seundo defina "MEU_AGENTE = AGENTE_2"

Em sequência rode o arquivo connect4.py

Todo resultado de uma jogada será escrito em um unico vetor no arquivo "saida_tabuleiro.txt",
basta copiar o resultado no executavel de outro grupo e digitar um input no terminal com o mesmo formato
para continuar o jogo.
